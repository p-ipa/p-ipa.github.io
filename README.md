# pipa.github.io
P-IPA : Prescription de l’infirmier(e) en pratique avancée Outil de recherche des DCI autorisées à la prescription par les IPA selon l’annexe VI de l’arrêté du 25 avril 2025 modifiant l’arrêté du 18 juillet 2018 fixant les listes permettant l’exercice IPA (article R.4301-3 du CSP).
